```
│  ║                                                   ║  │
│  ║     ┌───────────────────────────────────────┐    ║  │
│  ║     │                                       │    ║  │
│  ║     │   BLOG                                │    ║  │
│  ║     │                                       │    ║  │

```

## Размеры
- Ширина: 1200px
- Высота: 630px
- Формат: PNG

## Элементы
1. **Категория** (вверху) - BLOG / NEWS / TUTORIAL
2. **Заголовок** (центр) - Название поста (до 3 строк)
3. **Название блога** (внизу) - Muhammadali Blog
4. **Иконка** (справа внизу) - Круглая иконка

## Цвета
- Фон: Градиент #667eea → #764ba2 → #f093fb
- Контейнер: Белый с прозрачностью
- Текст: Темно-серый (#1a202c)
- Категория: Синий (#667eea)

## Где показывается
✅ Telegram - большое превью

✅ Facebook - превью поста
